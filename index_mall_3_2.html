<신규상품 등록>

[관리정보]
① 거래처 코드 : "AP-○○○○○"형식으로 표시/등록 됩니다. (등록된 정보는 변경/수정할 수 없습니다.)
② 상품타입 : 업체등록시 최초 등록된 거래방식으로 표시/등록됩니다.
③ 회원제 노출여부 : 몰관리자에서 삭제
④ 상품코드 : 상품등록시 자동설정 되며 개별적으로 적용 불가능 합니다.
  ※ 단! 상품 일괄등록시는 제외 입니다.
⑤ 카테고리 : 제품이 해당되는 카테고리를 중복으로 등록할 수 있습니다.
⑥ 판매기간 설정 : 설정된 기간 동안만 판매 가능하며, 설정된 종료일 이후에는 판매되지 않습니다.
  ※ 일시 판매중지 처리하실 경우, 종료일을 현재날짜 이전의 과거 날짜를 넣어주시면 됩니다.
⑦ 판매여부
  ※ 전시 : 상품이 정상적으로 노출 됩니다.
  ※ 품절 : 상품이 노출은 되지만 "품절" 이라는 문구가 표시 됩니다.
  ※ 단종 : 상품이 노출은 되지만 "단종" 이라는 문구가 표시 됩니다.
  ※ 중지 : 상품이 노출 되지 않습니다.
⑧ 승인여부 : 정책 정해진 이후에 프로세스 정립
  ※ 대기 :
  ※ 보류 :
  ※ 승인 : 승인요청시 몰관리자에게 상품승인요청이 됩니다.


[가격정보]
ⓙ 판매가격 : 쇼핑몰에 실제 판매할 가격을 입력합니다. 상품 등록 시 필수항목입니다.
② 과세구분 : 과세, 비과세 상품 여부를 선택합니다.
③ 매입원가(계약원가) : 제품의 공급 가격을 입력하세요.
④ 권장소비자가 : 제품의 할인전 판매가를 입력하세요.
⑤ 온라인최저가격 : 시중(온라인/오프라인)에서 판매되고 있는 최저가격을 입력하세요.


[기본정보]
① 브랜드 : 등록되어 있는 브랜드가 노출되며, 해당되는 브랜드를 선택하시면 됩니다.
  ※ 해당브랜드가 없을 경우 기타로 입력하시면 됩니다.
② 상품명 : 최대 255자까지만 입력 가능하며 한글,영문만 입력하세요! HTML은 입력하시면 안됩니다.
③ 자사상품코드 : 귀사에서 사용중인 상품코드를 입력하세요.
  ※ 용도 : 귀사에서 등록한 상품에 대한 자체관리용
④ MD MENTION : 제품을 홍보할 수 있는 보조문구를 입력하세요.
⑤ 상품검색용 Hash Tag : 상품과 관련된 단어를 입력해주세요.
  ※ 단어와 단어 사이는 콤마 ( , ) 로 구분하여 여러개를 입력할 수 있습니다. 예시) 빨강, 노랑, 파랑
⑥ 제조사 : 제품을 제조한 회사명을 입력해주세요. 예시)삼성, 현대
⑦ 유통기한 : 명절상품 등 단기 운영되는 상품의 경우 유통기한을 입력해주시기 바랍니다.
⑧ 생산국(원산지) : 제품의 생산(원산)지를 입력하세요. 예시) 한국, 중국
⑨ 비고: 쇼핑몰에 노출되는 부분이 아니며 관리용으로만 사용 됩니다.
⑩ 상품영상(규격영상) : 추가개발이후 메뉴변경예정


[기본정보 - 상품 이미지 및 상세정보]
※ 리스트용 이미지, 상품상세이미지, 상품확대이미지 등 각각 필요한 상품이미지를 등록합니다.
※ 상품이미지는 (.jpg / .gif / .png)파일만 올릴 수 있습니다.

[이미지 등록방식]
　 ※ 직접 업로드 : PC에서 파일찾기를 통해 직접 업로드하는 방식 입니다.
　 ※ URL 입력 : 외부 이미지를 http://를 포함해 입력하는 방식 입니다. URL방식은 쇼핑몰 로딩속도가 다소 늦어질 수도 있습니다. [비권장]

[자동리사이즈]
  ※ 확대(원본)이미지만 올리면 자동으로 리사이즈 되는 편리한 기능입니다.
  　 a. 확대(원본)이미지(가로세로 450픽셀이상 권장)를 등록한 후 "리사이즈 사용"에 체크하세요.
  　    다른 이미지들은 올리지 않으셔도 됩니다. 자동으로 사이즈가 조정되어 저장됩니다.
  　 b. 반드시 상품확대이미지(원본이미지)를 가로넓이가 450픽셀이상으로 넣어야만 높은 퀄리티로 리사이징됩니다.
　　  움직이는 Gif (Ani Gif) 이미지는 자동리사이징이 안됩니다. 리사이징을 시키면 스톱된 이미지로 리사이징됩니다.
  ※ [소 이미지 ~ 대 이미지5번] 이미지 등록시 "리사이즈 사용"에 체크없이 저장 하실 경우 원본 사이즈 그대로 업로드 됩니다.

[소 이미지] : 쇼핑몰메인, 서브페이지에서 노출 되는 리스트용 이미지입니다. 필수등록!
[중 이미지1] : 상품상세페이지에서 노출 되는 이미지입니다. 필수등록!
[중 이미지2,3,4,5] : 상품상세페이지에서 노출 되는 이미지이며 미리보기용으로 사용 됩니다.
[상세설명] : 내 쇼핑몰에 노출될 상세한 상품 설명 내용을 입력하세요. 다양한 각도의 상품사진을 곁들이면 더욱 효과적입니다.


[기본정보 - 상품 필수정보]
※ 공정거래위원회에서 공고한 전자상거래법 상품정보제공 고시에 관한 내용을 필독해 주세요! 바로가기
※ 전자상거래법에 의거하여 판매상품의 필수(상세)정보 등록이 필요합니다.
①번 항목에 SELECT 박스를 선택 하시면 ②번 항목에 해당 값이 자동으로 위치 합니다.
 예시)1. 제품 소재, 2. 색상, 3. 치수 등..

[기본정보 - 관련상품]
① 에서 기등록된 상품을 텍스트로 입력하여 검색하면 ② 번에 관련상품이 출력되고, 추가시 ③ 번에 관련상품등록이 완료됩니다.
※ 등록된 관련상품은 제품상세페이지 최하단에 리스트되어 보여집니다.


[옵션정보]
주문조건 : 소비자회원이 주문옵션을 반드시 선택해야 주문이 가능한 필수항목 입니다.
  ※ 용도 : 본 상품에 대한 다양한 타입 및 가격 설정 예시) 티셔츠 타입 A, 타입 B, 타입 C
  ※ 구매수량 : 구매수량 데이터에 반영 됩니다. 주문금액 계산 : (본 상품금액 + 추가금액) × 수량
  ※ 부분취소 여부 : 주문옵션별 부분취소/반품이 가능 합니다.
① 옵션1,2,3 항목에 콤마로 구분하여 옵션 소분류를 등록한 후 ②번 "옵션목록생성" 버튼을 클릭합니다. 
③ 옵션이 자동으로 생성되어 배열 됩니다.
④ 전체 옵션의 추가금액, 재고/통보수량 및 사용여부를 일괄 적용할 수 있습니다. 단, 체크된 항목만 적용됩니다.
  ※ 재고수량 무제한 : 상품주문시 재고가 차감 되지 않습니다.
  ※ 통보수량 무제한 : 상품관리 > 상품 옵션재고관리에 노출 되지 않습니다.
⑤ "옵션일괄적용" 버튼을 클릭하시면 ④번에 체크 된 항목만 일괄적으로 적용 됩니다.

[상품 추가옵션]
  ※ 용도 : 본 상품의 액세서리 및 코디상품을 ‘추가’ 판매 예시) 카메라 삼각대, 케이스, 메모리카드
  ※ 구매수량 : 구매수량 데이터에 반영되지 않습니다. 
     주문금액 계산 : (본 상품금액 × 수량) + (추가구성 금액 × 수량)
  ※ 부분취소 여부 : 주문옵션별 부분취소/반품이 불가 합니다.
  ※ "옵션추가" 버튼을 클릭하시면 등록 가능한 옵션 항목이 무한 생성 됩니다.
  ※ 이하 작동 원리는 위 주문옵션과 동일 합니다.


[물류정보]
① "몰관리 > 배송지등록"에서 기등록한 배송지를 선택하여 등록합니다.
② 입고/출고단위를 EA 또는 Box로 표시합니다.
③ 무제한 : 재고수와 상관없이 무제한 주문이 가능합니다. 주문시 수량 감소 안됨!
  한정 : 재고수를 체크하며 재고 부족시 주문이 불가능 합니다. 주문시 수량 감소!
④ 판매가능 수량을 입력합니다.
⑤ 상품의 재고가 통보수량보다 작을 때 상품 재고관리에 표시됩니다.
  옵션이 있는 상품은 개별 옵션의 통보수량이 적용됩니다. 설정이 무제한이면 재고관리에 표시되지 않습니다.
⑥ 최소 : 1회 주문 시 최소주문수량을 입력합니다. 한번에 여러개를 판매해야 하는 상품일 경우 입력을 해주세요.
  최대 : 최소주문수량과는 반대로 1회 주문 시 가능한 최대 주문량을 입력합니다.
    ※ 최소, 최대 둘중 하나만 입력도 가능하며 최소에 값이 있을 경우 수량에 기본적으로 고정되어 표시 됩니다.

  이하 주문수량은 주문 불가!
⑦ 1인주문 횟수를 제한할 경우 선택 및 입력합니다.
⑧ 제품의 바코드번호를 입력합니다.
⑨ 무료배송 : 배송비가 부과되지 않습니다.
   유료배송 : {개본배송비}원을 주문금액 또는 수량에 상관없이 동일 주문건에 배송비를 한번만 부과합니다.
   조건배송수량 : {개본배송비}원의 배송비를 부과하며 단! 주문금액이 {조건배송비}원 이상이면 무료배송 처리됩니다.
⑩ 도서 산간 추가배송비 발생시 입력해주시기 바랍니다.

[배송정보/규격]
  ■ 상품에 대한 규격 및 수출입상품일 경우 입력해주시기 바랍니다.
