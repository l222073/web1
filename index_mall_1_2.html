● 브랜드 등록요청의 용도 : 상품등록을 목적으로 브랜드 조회를 했으나 브랜드가 없을 경우 거래처가 직접 브랜드를 등록 요청하는 메뉴
● 브랜드 등록요청 방법


	 ① 브랜드 등록요청 선택 : 몰관리 > 브랜드 등록요청

	 ② 브랜드명 KOR 작성 : 한국어로 브랜드명 작성

	 ③ 브랜드명 ENG 작성 : 영문으로 브랜드명 작성

	 ④ 브랜드 로고 파일선택 : 파일로만 브랜드 이미지를 등록 할 수 있음 사이즈(128픽셀 * 40픽셀)

	 ⑤ 등록 할 브랜드 로고파일 선택 : 등록 할 파일 선택

	 ⑥ 등록 파일 최종 선택

	 ⑦ 선택 된 파일명 다시 확인
